# ML-Financial-Health-Analytics
Machine Learning Project for Supervised Stroke Prediction and Unsupervised Credit Card Clustering.

# 🚀 AWS End-to-End Machine Learning Lifecycle Project
**A dual-track technical exploration of Supervised and Unsupervised Learning.**

This project implements a full-cycle machine learning pipeline to solve two distinct analytical challenges: **Stroke Prediction** (Supervised Learning) and **Credit Card User Segmentation** (Unsupervised Learning). The project demonstrates a professional transition from raw data exploration to cloud-integrated model optimization using AWS SageMaker.

## 📌 Project Overview
The goal of this investigation is to demonstrate proficiency across the entire ML lifecycle:
* **Supervised Track:** Utilizing clinical patient data to predict the likelihood of a stroke (Classification).
* **Unsupervised Track:** Discovering hidden patterns and behavioral segments within credit card user data to facilitate targeted financial strategies (Clustering).

---

## 🚀 Key Technical Phases

### Phase 1: Statistical Exploration & Data Engineering
* **EDA:** Performed comprehensive Exploratory Data Analysis using Heatmaps, Box Plots, and Correlation Matrices to understand feature relationships.
* **Statistical Analysis:** Calculated core numerical measures (mean, median, std dev) to assess sample variability and data integrity.
* **Pre-processing:** Implemented robust data cleaning (handling missing values), one-hot encoding for categorical variables, and feature scaling to ensure model readiness.

### Phase 2: Multi-Model Development
* **Supervised Learning:** Built and compared classification models (e.g., Random Forest vs. Support Vector Machines) to predict stroke risk.
* **Unsupervised Learning:** Applied clustering techniques (e.g., K-Means and PCA) to identify distinct customer segments based on spending behavior.

### Phase 3: Cloud Optimization & Evaluation
* **AWS Integration:** Leveraged AWS SageMaker for accelerated model training and hyperparameter tuning.
* **Performance Tuning:** Utilized Grid Search and K-Fold Cross-Validation to ensure model robustness and prevent overfitting.
* **Critical Evaluation:** Analyzed efficiency using ROC curves and Confusion Matrices for supervised tasks, and Silhouette scores for cluster validation.
